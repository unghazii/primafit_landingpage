Tentang Primafit

Primafit adalah aplikasi kesehatan personal komprehensif yang dikembangkan untuk membantu pengguna mengelola kesehatan secara holistic dengan berbagai fitur canggih dan mudah digunakan.

Aplikasi ini dikembangkan dengan menggunakan framework Flutter dengan database SQLite untuk penyimpanan data lokal dan dilengkapi dengan algoritma diagnosa menggunakan dart similarity string.

Data Pribadi Aman

Data kesehatan pribadi disimpan secara lokal dan tidak dibagikan tanpa persetujuan pengguna.

Penggunaan Offline

Aplikasi dapat digunakan tanpa koneksi internet karena data disimpan secara lokal di perangkat.

Update Berkala

Pembaruan rutin dengan fitur baru dan peningkatan performa untuk pengalaman terbaik.
